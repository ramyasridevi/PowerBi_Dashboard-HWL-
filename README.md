# 📊 Healthcare Dashboard – Power BI

## 📝 Overview
This project is an interactive healthcare dashboard built using Power BI to analyze patient data across Outpatient, Inpatient, and Day Care categories. It helps users quickly understand trends, distributions, and key insights.

## 🎯 Objective
- Analyze patient trends over time  
- Understand specialty-wise distribution  
- Provide interactive and easy-to-use insights  

## 🏥 Patient Types
- **Outpatient:** Visit and leave the same day  
- **Inpatient:** Admitted and stays overnight  
- **Day Care:** Treatment without overnight stay  

## 📈 Dashboard Features

### 🔹 Trend Analysis (Line Charts)
Displays how patient numbers change over time across different case types.

### 🔹 Distribution Analysis (Donut Chart)
Shows how patients are distributed across different specialties.

### 🔹 KPI & Multi-Row Cards
Highlights key metrics for quick understanding of overall performance.

### 🔹 Interactive Tooltips
Hover over visuals to see detailed breakdowns like specialty-wise distribution.

### 🔹 Navigation & Drill-Through
Buttons allow smooth movement between Summary and Detail pages.

### 🔹 Dynamic Filtering
Slicers enable filtering data based on time periods for better analysis.

## 🛠️ Tools & Technologies
- Power BI  
- DAX  
- Data Modeling  

## 💡 Key Highlights
- Implemented custom tooltip for drill-down on hover  
- Built clean and user-friendly UI  
- Solved data modeling and relationship issues

## 🚀 How to Use
1. Open the `.pbix` file  
2. Explore Summary page  
3. Use filters to adjust time period  
4. Hover over charts for deeper insights  
5. Navigate to Detail page
